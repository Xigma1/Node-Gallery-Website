# Node-Gallery-Website

Requirements :
- Node
- Nodemon
- Body Parser
- ejs
- uuid
- multer
- sharp

To set up :
- Clone repo
  - Run `npm install express body-parser ejs uuid multer sharp`
- Run `npm install nodemon`
- To launch open CMD or Node, then run `npm start`
